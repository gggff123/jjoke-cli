## What is this
**The Jjoke-CLI** is an app['module'] that fetches a joke from the **[Jokes API](https://sv443.net/jokeapi/v2/)** all on your cli available for free!

## How to run it
+ first clone the repo by using | ``git clone https://github.com/gggff123/jjoke-cli.git``.
+ Then download the dependencies by using | ``npm install @inquirer/prompts chalk figlet gradient-string``.|
+ And then run the file by |``npm run app.js``.

## Requirements
* make sure you have **Node.js** installed or this project will not **Work**.
* to install nodejs you can go to their officaial website **[Node.js](https://nodejs.org/en)**

## For those who dont want to do all by themselves 
+ you can download the pre-build as a module by using ``npm i jjoke-cli``.
